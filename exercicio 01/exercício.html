Como fazer a configuração incial do git bash

Instalação do git bash

Entre no link https://git-scm.com/downloads e selecione a opção Download for windows e
selecione 64 bit. Após ter instaldo no pc você irá executa-lo, clicando duas vezes no mesmo
Com o programa aberto, clique em next, e depois em instal.

Antes de começar a importar coisas do github precisamos fazer a configuração inicial.

Configuração inicial do git bash

A primeira coisa que você deve fazer ao instalar Git é configurar seu nome de usuário e endereço de e-mail.
Isto é importante porque cada commit usa esta informação, e ela é carimbada de forma imutável nos commits que você começa a criar:


Utilize os seguinte códigos:
git config --global user.name "Fulano de Tal"
git config --global user.email fulanodetal@exemplo.br


O primeiro código é utlizado para colocar seu nome de usuário, e o segundo código é utilizado para linkar seu email do github.

Testando as configurações

Se quiser testar as suas configurações você pode utilizar o código git config --list
para listar as suas configurações.
